## About Installation

### Requirements
`php` > 8.0

`composer`>= 2.4

### Setup instruction 
`git clone [git repo]`

`git checkout main`

`git pull origin main`

`cd project folder`

`composer install`

`cp .env.example .env `

#### Set DB Connection Configs (.env)

`php artisan key:generate`

`php artisan migrate`
#### Test Server
`php artisan serve` test call api http://127.0.0.1:8000/api/visitors Get
